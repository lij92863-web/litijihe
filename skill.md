# 轨迹划痕要求

不要一开始显示完整轨迹。

轨迹必须随着红点拖动逐渐留下。

当红点靠近本题真实有效轨迹时，自动吸附并留下划痕。

有效轨迹只有三段：

1. 上底面轨迹：

   * z = 2
   * x² + y² = 1
   * 0 ≤ x ≤ 1
   * 0 ≤ y ≤ 1

2. 侧面 x = 1 轨迹：

   * x = 1
   * y² + z² = 4
   * 0 ≤ y ≤ 1
   * 0 ≤ z ≤ 2

3. 侧面 y = 1 轨迹：

   * y = 1
   * x² + z² = 4
   * 0 ≤ x ≤ 1
   * 0 ≤ z ≤ 2

实现方法：

1. 用参数方程采样三段轨迹；
2. 每段采样至少 200 个点；
3. 拖动红点时计算红点到三段轨迹采样点的最近距离；
4. 如果距离小于吸附阈值，就把红点吸附到最近轨迹点；
5. 同时把该采样点附近若干点标记为 visited；
6. 渲染时只画 visited 过的轨迹点或连续轨迹段；
7. 未 visited 的轨迹不能显示。

吸附阈值建议为 0.12 到 0.18，需要保证课堂拖动容易触发，但不能误触发太严重。
